Problem statement  <br>
- Create a simple command-line rock-paper-scissors game where a user competes against the computer, testing basic programming concepts like input handling, random choices, and conditional logic.

Scope of the project  <br>
- The project implements the core mechanics of the game with user input, random computer choice, and outcome calculation. It does not include advanced features like multiplayer support, score tracking, or UI enhancements.

Target users  <br>
- Beginners learning Python programming who want a hands-on project to practice using modules, conditionals, and user input; casual users who enjoy playing classic games through the console.

High-level features  <br>
- Random selection of computer's move using the random module  
- Case-insensitive user input for rock, paper, or scissors  
- Logic to determine and display the winner or declare a tie  
- Clear text-based interaction suitable for command line execution
